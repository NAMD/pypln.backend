Module cmanager

source code

This executable is the manager of a pypln cluster. All tasks should be started through it and are monitorable/controllable through it.

Cluster configuration must be specified on a config file pypln.conf with at least the following sections: [cluster] master_ip = 127.0.0.1 nodes = ['127.0.0.1', '10.250.46.208'] # list of IPs to add to PyPLN cluster [cluster]
